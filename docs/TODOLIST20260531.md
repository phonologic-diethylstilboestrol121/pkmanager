  - 利用 PostgreSQL JSONB GIN 索引加速
  - 支持筛选：物种 / 闪光 / 性格 / 特性 / 持有物 / 球种 / 来源版本 / 性别 / 等级范围 / IV/EV 下限 / 招式(Any/All) / OT名称/TID / 合法状态

  - 拖拽MoveSlot/SwapBoxes修复（同箱temp变量/跨箱分数组）
