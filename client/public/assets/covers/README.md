# 游戏封面图目录

请将宝可梦游戏封面图放置在此目录下，文件名格式为 `{gameId}.png`（支持 .png / .jpg / .webp）。

## 命名对照表

| 文件名 | 游戏 |
|--------|------|
| `pkm_ruby.png` | 宝可梦 红宝石 |
| `pkm_sapphire.png` | 宝可梦 蓝宝石 |
| `pkm_emerald.png` | 宝可梦 绿宝石 |
| `pkm_firered.png` | 宝可梦 火红 |
| `pkm_leafgreen.png` | 宝可梦 叶绿 |
| `pkm_diamond.png` | 宝可梦 钻石 |
| `pkm_pearl.png` | 宝可梦 珍珠 |
| `pkm_platinum.png` | 宝可梦 白金 |
| `pkm_heartgold.png` | 宝可梦 心金 |
| `pkm_soulsilver.png` | 宝可梦 魂银 |
| `pkm_black.png` | 宝可梦 黑 |
| `pkm_white.png` | 宝可梦 白 |
| `pkm_black2.png` | 宝可梦 黑2 |
| `pkm_white2.png` | 宝可梦 白2 |

## 推荐尺寸

- GBA 封面：~500×500 或更大
- NDS 封面：~500×450 或更大

图片会自动缩放适配卡片。若无封面图，卡片会显示默认图标。
